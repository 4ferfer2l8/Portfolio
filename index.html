Projetos Recentes

SGRC - Trabalho de Conclusão de Curso Técnico

O SGRC é uma aplicação web desenvolvida para a empresa de fibra óptica Estrategic Engenharia, com o objetivo de automatizar os processos de validação de relatórios técnicos que antes eram realizados manualmente pelos colaboradores. O sistema foi desenvolvido como Trabalho de Conclusão de Curso (TCC) do curso técnico em Desenvolvimento de Sistemas no SENAI CIMATEC.

Tecnologias utilizadas: HTML, CSS, JavaScript, Node.js, PostgreSQL e Google Cloud.

Demo

Rep&Aprov - TCC Técnico de Mecânica

O Rep&Aprov é uma aplicação web desenvolvida em parceria com uma equipe do curso técnico de Mecânica, como parte do Trabalho de Conclusão de Curso (TCC) deles. O projeto consiste em um quiz interativo sobre a manutenção de bombas centrífugas, com o objetivo de tornar as aulas mais dinâmicas e engajadoras.

Tecnologias utilizadas: React.js com Tailwind CSS.

Jogo de Plataforma Invasão da SunnyLand

SunnyLand é um jogo 2D de plataforma desenvolvido como projeto acadêmico. Nele, o jogador controla a personagem Foxy em uma jornada por diferentes fases, enfrentando inimigos e chefes até alcançar o desafio final. Ao derrotar o chefão, o jogador conclui o jogo com êxito.

Tecnologias utilizadas: Unity com programação em C#.
